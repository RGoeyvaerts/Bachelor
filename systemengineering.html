Programmable Robots

  1. Home
  2. Programmable Robots

Project information

  • Category: Electronics, Programming
  • Client: Thomas More
  • Project date: 31 January, 2023

Brief Summary

We were assigned by our teacher to spice up math through Soccer. We then designed a robot that uses simple electronic components whose values can be calculated. We use frames made with a 3d printer. In addition to this, programming the robot also adds to the mathematical aspect. The robots are programmed in pictoblox a project of stempedia which also has its own app with a bluetooth controller with which we control the robots.
